# Decision: Use Generated Boradori Brand Boards

Date: 2026-04-26

## Status

Accepted

## Context

The user provided two generated brand boards for 맥락 보라돌이. They define the mascot tone, color palette, context knot motif, and product language: 모으다, 이해하다, 압축하다, 연결하다.

## Decision

Use the images as public brand assets and align the app UI with the palette and language from the boards. Keep the app itself product-first, with the full brand boards shown as a supporting brand section and README assets rather than replacing the core paste/compress/export workflow.

## Consequences

- The demo now feels more distinctive and hackathon-ready.
- README has immediate visual identity material.
- The product UI remains usable while inheriting the mascot direction.
- Future asset work can crop or regenerate smaller mascot-only images if needed.
